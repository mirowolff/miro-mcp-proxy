Miro MCP Proxy

MCP SSE to stdio proxy for Miro Design System

curl -fsSL https://mirowolff.github.io/miro-mcp-proxy/install.sh | bash
View on GitHub

Note: You'll need to generate your access token at miro.design/mcp/token

What this does

  • 1 Downloads the proxy script to ~/.miro-mcp-proxy.sh
  • 2 Prompts you for your Miro access token and email
  • 3 Automatically configures Claude Desktop
  • 4 Ready to use after restarting Claude Desktop
