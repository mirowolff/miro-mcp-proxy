One command to install and configure the MCP proxy for Claude Desktop

❯ curl -fsSL https://mirowolff.github.io/miro-mcp-proxy/install.sh | bash

Copy and paste this command into your terminal, then press Enter to begin installation.

What this does

1

Downloads proxy and installs to ~/.miro-mcp-proxy.sh

2

You'll be asked to enter your email and access token.
Generate token →

3

Configures Claude Desktop

Done, ready to use! Quit (⌘ + Q) and reopen Claude Desktop
